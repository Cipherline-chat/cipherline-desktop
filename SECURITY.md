# Security Policy

This repository doesn't have the desktop client's source code published yet — but if
you've found a security issue with the live Cipherline service in the meantime, we
still want to hear about it.

**Please report privately to [security@cipherline.com](mailto:security@cipherline.com).**
Do not open a public issue for a security vulnerability.

See [cipherline.chat/security-disclosure](https://cipherline.chat/security-disclosure)
for our full disclosure policy, response commitments, and safe-harbor terms. A complete
`SECURITY.md` will land in this repo alongside the client source.
